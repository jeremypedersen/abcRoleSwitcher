Configuration

Simply enter your RAM role information into the Configuration box to the left. Target roles can be expressed with a role_arn or with both abc_account_id and role_name.

Optional parameters

  • color - The RGB hex value (without the prefix '#') for the color of the header bottom border and around the current profile.
    • [role1]
role_arn = arn:abc:iam::123456789012:role/marketingadmin
color = ffaaee

[role2]
abc_account_id = 987654321988
role_name = my_role
color = ffaaff
